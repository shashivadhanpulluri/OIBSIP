# ATM Interface (Java Console Application)

## Project Overview

This project is a console-based ATM Interface developed using Java and Object-Oriented Programming (OOP). It simulates the basic functionalities of an ATM system, allowing users to log in securely using a User ID and PIN and perform banking operations.

## Features

- Secure Login using User ID and PIN
- Maximum 3 Login Attempts
- Deposit Money
- Withdraw Money
- Transfer Money to Another Account
- Transaction History
- Balance Validation
- Console-Based Menu
- Object-Oriented Design

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class
- VS Code

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Navigate to the src folder.

```
cd src
```

4. Compile the project.

```
javac *.java
```

5. Run the project.

```
java Main
```
